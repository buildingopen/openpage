OpenPage

Open-source website builder where JSON is the source of truth. Visual drag-and-drop editor, AI generation, standalone HTML export.

Live Demo GitHub Repo

Why JSON-First?

Code generators (Lovable, v0, Bolt) are fragile: LLMs hallucinate imports, break builds, produce unmergeable diffs. Visual editors (Framer, Webflow) lock you into proprietary formats that AI agents can't read. OpenPage puts a typed JSON config between the editor and the renderer, so humans drag blocks while agents make surgical edits via API. Same source of truth, clean diffs, predictable results.

Visual Editor

Drag-and-drop blocks with real-time preview, inline editing, and block reordering

AI Generation

POST a prompt, get a complete website as structured JSON via Gemini

19 Block Types

Self-Hosted

Quick Start

git clone https://github.com/buildingopen/openpage.git
cd openpage
npm install
npm run dev

The editor opens at http://localhost:5173. Create a site, drag blocks, edit content, export to HTML.
